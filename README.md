# Gerenciador de Tarefas — GDG Londrina / Tech Week

Projeto desenvolvido como avaliação prática do 1º Bimestre — ADS/ESOFT 3º Semestre — Unicesumar.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas dinamicamente
- Remover tarefas individualmente
- Concluir tarefas (texto tachado + fundo verde)
- Contadores de tarefas pendentes e concluídas em tempo real
- Layout responsivo para telas pequenas

## Tecnologias

- HTML5
- CSS3 (Flexbox + Media Queries)
- JavaScript (manipulação de DOM)

## Como rodar

Abra o arquivo `index.html` no navegador.

## Testes

```bash
npm install
npm test
```
